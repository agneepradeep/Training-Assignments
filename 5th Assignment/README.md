## Overview
This project analyzes sales performance using Power BI. It includes sales trends,
profitability, target comparison, and product-level insights.

## Tools Used
- Power BI Desktop
- Power Query
- DAX
- Excel

## Data
- Raw sales data provided in Excel format
- Data modeled using a star schema with a calendar table

## Features
- Total Sales, Profit, Quantity
- YTD, LY, YoY Growth
- Target vs Actual analysis
- Interactive dashboards with slicers

## How to Open
1. Download the PBIX file
2. Open using Power BI Desktop
3. Refresh data if prompted
